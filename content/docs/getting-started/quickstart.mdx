---
title: 5-minute quickstart
description: Accept a Stripe test payment and unlock a feature in a React app.
---

This is the shortest complete path to a first test payment. It uses Stripe test
mode, anonymous credit-code wallets, one credit pack, and a React app with
shadcn already initialized.

## 1. Create a test project

In the dashboard, connect a **Stripe sandbox** key. vantezzen/pay validates the
key and creates the webhook endpoint automatically. If your restricted key
cannot create webhooks, the wizard shows the one manual fallback step.

Create a credit-pack product, then add a one-time price such as **100 credits
for $5**. Copy the publishable key from the project’s **Developers** tab.

## 2. Install the app components

```bash
npx shadcn@latest add \
  https://pay.vantezzen.io/r/pay-provider.json \
  https://pay.vantezzen.io/r/pay-purchase.json \
  https://pay.vantezzen.io/r/pay-gate.json
```

Set your project’s values:

```bash title=".env.local"
NEXT_PUBLIC_PAY_URL=https://pay.vantezzen.io
NEXT_PUBLIC_PAY_KEY=pay_pk_your_key
```

Use your own deployment URL when self-hosting.

## 3. Wrap your app

```tsx title="app/layout.tsx"
import { PayProvider } from "@/components/pay/provider";

export default function RootLayout({ children }: { children: React.ReactNode }) {
  return (
    <html lang="en">
      <body>
        <PayProvider
          baseUrl={process.env.NEXT_PUBLIC_PAY_URL!}
          publishableKey={process.env.NEXT_PUBLIC_PAY_KEY!}
        >
          {children}
        </PayProvider>
      </body>
    </html>
  );
}
```

## 4. Add one paid action

```tsx title="components/generate-button.tsx"
"use client";

import { CreditGate } from "@/components/pay/credit-gate";
import { useCredits } from "@/components/pay/provider";
import { Button } from "@/components/ui/button";

export function GenerateButton() {
  const { deduct } = useCredits();

  async function generate() {
    await deduct(1, { idempotencyKey: crypto.randomUUID() });
    // Run the paid work here.
  }

  return (
    <CreditGate cost={1}>
      <Button onClick={generate}>Generate</Button>
    </CreditGate>
  );
}
```

Open the app, select the purchase option, and pay with Stripe’s test card
`4242 4242 4242 4242` using any future date and CVC. The popup closes when
payment is confirmed and the button unlocks automatically.
